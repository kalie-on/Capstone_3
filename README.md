## Capstone_3 – EmporiUm Sales Analysis with Power BI
## By Ndifon Caleb Ndakina
## West Region
## Overview

As a Data Analyst for EmporiUm's West Region, I will analyze four years of sales data to evaluate business performance across the region. Using Power BI, I will clean, transform, model, analyze, and visualize the data to uncover meaningful insights that support regional directors, territory managers, and store managers in making informed business decisions.

My report will focus on sales trends, category performance, state-level sales distribution within the West Region, and the top-selling general audience books and their authors.

## Objectives
- Clean and prepare raw sales data using Power Query.
- Build an efficient data model with appropriate relationships.
- Create interactive Power BI dashboards.
- Identify sales trends over time.
- Analyze sales performance by product category.
- Compare sales across states within the assigned region.
- Identify the top-selling books and their authors (excluding textbooks).
- Present business insights that support better decision-making. 

## Tools & Technologies Used
- Power BI Desktop
- Power Query
- DAX (Data Analysis Expressions)
- Git $ GitHub
- Power BI Service
- Microsoft PowerPoint (presentation)
